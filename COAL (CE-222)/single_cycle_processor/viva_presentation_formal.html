Project Overview

  • Development of a 32-bit RISC-V CPU core.
  • Implementation based on the Single-Cycle design methodology.
  • Execution of a custom 7-instruction subset of the RV32I ISA.
  • Verified through hardware-level simulation and timing analysis.

Processor Architecture

Key Modules:

  • IF: PC & Instruction Memory
  • ID: Decoder & Register File
  • EX: ALU & Immediate Generator
  • MEM: Data Memory Interface
  • WB: Result Multiplexing
